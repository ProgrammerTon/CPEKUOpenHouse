# โครงงานเปิดบ้านวิศวะคอมมหาวิทยาลัยเกษตรศาสตร์จำลอง (CPEKU Open House Simulator)

โครงงานนี้เป็นแอปพลิเคชันที่ถูกพัฒนาขึ้นเพื่อเผยแพร่ข้อมูลและเตรียมความพร้อมสำหรับผู้เข้าสนใจศึกษาต่อภาควิชาวิศวกรรมคอมพิวเตอร์ มหาวิทยาลัยเกษตรศษสตร์บางเขน โดยประกอบด้วย 3 ฟีเจอร์หลัก ดังนี้

---

## ฟีเจอร์ที่ 1: แอปพลิเคชันเดินสำรวจอาคาร

<!--![Building Exploration](images/feature1.png) -->

แอปพลิเคชันนี้พัฒนาขึ้นโดยใช้เทคโนโลยี **Unity Engine** และเขียนควบคุมกลไกต่าง ๆ ด้วยภาษา **C#**  
เพื่อจำลองสภาพแวดล้อมภายในอาคารจำนวน **8 ชั้น**

ผู้ใช้งานสามารถ:

- เดินสำรวจพื้นที่ภายในอาคารได้อย่างอิสระ  
- โต้ตอบกับวัตถุต่าง ๆ ภายในฉาก  
- ดูข้อมูลของจุดที่น่าสนใจในแต่ละชั้น  

---

## ฟีเจอร์ที่ 2: เว็บไซต์ศูนย์รวมข้อมูลรายวิชา

<!--![Course Website](images/feature2.png) -->

เว็บไซต์นี้พัฒนาด้วย:

- **Frontend:** JavaScript และ React  
- **Backend:** Node.js  
- **Database:** MongoDB  

โดยแบ่งการใช้งานเป็น 2 ระดับ:

### ผู้ใช้งานทั่วไป
- ค้นหาและดูข้อมูลรายวิชา
- เล่นเกมเสริมสร้างทักษะการคิด
  
### ผู้ดูแลระบบ
- เพิ่มข้อมูลรายวิชา
- แก้ไขข้อมูลรายวิชา
- จัดการข้อมูลภายในระบบ

---

## ฟีเจอร์ที่ 3: ระบบแชทบอทอัจฉริยะ 

<!-- ![Chatbot](images/feature3.png) -->

ระบบแชทบอทพัฒนาด้วยภาษา **Python** โดยใช้เทคนิค:

- Retrieval-Augmented Generation (RAG)
- Vector Database
- PDF Processing Library

ความสามารถของระบบ:

- ตอบคำถามเกี่ยวกับคณะวิศวกรรมคอมพิวเตอร์
- ดึงข้อมูลจากเอกสาร PDF
- ให้คำตอบที่ถูกต้องตามข้อมูลจริง

---

## เทคโนโลยีที่ใช้

- Unity Engine
- C#
- React
- Node.js
- MongoDB
- Python
- RAG
- Vector Database
