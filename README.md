ฟีเจอร์ที่ 1 แอปพลิเคชันเดินสำรวจอาคารภาควิชาวิศวกรรมคอมพิวเตอร์ พัฒนาขึ้นโดยใช้เทคโนโลยี Unity Engine และเขียนควบคุมกลไกต่าง ๆ ด้วยภาษา C# เพื่อจำลองสภาพแวดล้อมภายในอาคาร 8 ชั้น ผู้ใช้งานสามารถสำรวจพื้นที่ภายในอาคารได้ตามใจชอบ พร้อมทั้งมีจุดทำปฏิสัมพันธ์กับวัตถุต่าง ๆ ที่จัดวางไว้ตามฉากเพื่ออธิบายข้อมูลของจุดที่น่าสนใจในแต่ละชั้น
